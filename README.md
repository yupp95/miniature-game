## miniature-game
웹브라우져로 미니 게임 만들기🎮

**Canvas 기능을 활용한 미니 게임 만들기**<br>

**🤔Canvas**<br>
- HTML5의 <canvas>엘리먼트는 단지 그래픽들을 위한 컨테이너이다.
- 그래픽을 그리기 위해 자바스크립트를 사용해야 한다.
- IE8은 지원이 안됨.
